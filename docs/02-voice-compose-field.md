# VoiceComposeField Component

## What it does

Adds **voice-to-text** capability to text fields in the admin panel. Speak and your words become text!

## Location

`src/components/VoiceComposeField/index.tsx`

## Features

- **Speech Recognition** - Talk to type
- **20+ Languages** - Bengali, Hindi, Spanish, French, German, Arabic, Chinese, Japanese, Korean, and more
- **Text-to-Speech** - Hear your content read back
- **AI Compose** - Works with Payload's AI plugin

## How it works

1. Click the microphone button
2. Select your language
3. Start speaking
4. Text appears automatically
