# Blocks System

## What it does

Provides drag-and-drop content building blocks for flexible page layouts.

## Location

`src/blocks/`

## Available Blocks

### Content Block
Multi-column text layouts

| Column Size | Width |
|------------|-------|
| oneThird | 33% |
| half | 50% |
| twoThirds | 66% |
| full | 100% |

### Archive Block
Displays posts dynamically

- Filter by category
- Set item limit
- Choose collection

### Call-to-Action Block
Action buttons with text

- Rich text content
- Multiple links
- Appearance options (default/outline)

### Media Block
Image or video embed

- Required media upload
- Responsive display

### Form Block
Embedded forms

- Connect to form builder
- Optional intro text

### Banner Block
Notification messages

- Styles: info, warning, error, success
- Rich text content

### Code Block
Syntax-highlighted code

- Languages: TypeScript, JavaScript, CSS
- Code editor field

## Usage

Add blocks to page layouts in admin panel, arrange and configure as needed.
