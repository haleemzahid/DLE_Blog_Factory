# Plugins Overview

## What they do

Extend PayloadCMS with additional features.

## Installed Plugins

### Form Builder
`@payloadcms/plugin-form-builder`

Create contact forms, surveys, sign-ups.
- Drag-drop form fields
- Email notifications
- Submission storage

### Nested Docs
`@payloadcms/plugin-nested-docs`

Hierarchical content structure.
- Parent-child categories
- Breadcrumb navigation
- URL hierarchies

### Redirects
`@payloadcms/plugin-redirects`

URL redirect management.
- 301/302 redirects
- Bulk redirect import
- Link to pages/posts

### SEO
`@payloadcms/plugin-seo`

Search engine optimization.
- Meta titles
- Descriptions
- Social images
- Auto-generation

### Search
`@payloadcms/plugin-search`

Full-text search.
- Index posts
- Fast queries
- Field mapping

### Payload AI
`@ai-stack/payloadcms`

AI content generation.
- Smart compose
- Content suggestions

### Vercel Blob
`@payloadcms/storage-vercel-blob`

Cloud file storage.
- Scalable uploads
- CDN delivery

### Vercel Postgres
`@payloadcms/db-vercel-postgres`

Cloud database adapter.
- Managed Postgres
- Auto-scaling
